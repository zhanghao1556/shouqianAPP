## 待用户手工补充

以下字段不得根据其他产品类推；补齐前在内部接线图中显示“需复核”，不阻断其他已确认连线。

### 未知外接设备

- 录播主机和录播摄像机的具体项目接口：在已确认的三种 LINE IN 形式中由客户选择；若实物不属于三者，按设备型号专项复核。
- 其他尚未录入的外接设备：继续显示“接口形式需复核”，不得借用已确认设备的面板或接口名。

## Agent 询问格式

后续每次只问一个具体问题，优先让 Agent 回答“是 / 不是”。示例：`线阵拓展器输出1的面板原始标识是否为 MIC OUT 1？` 若回答“不是”或“不完全”，再追问实际标识与适用边界。Agent 与本文件用户确认口径冲突时，继续以用户口径为准并追加冲突记录。
