# 音翼接口接线专项审计

## 2026-07-20 DT2 Pro LINE OUT

- 用户确认：每组 `L / R / G` 中，`L`、`R` 是两路可独立分配的非平衡信号，可分别接两个通道或两个设备；两路信号在 DT2 Pro 端共用 `G`，两根地线并接在同一个 `G` 孔。
- 公司 Agent 证据：安装手册中的 DT2 Pro `LINE OUT` 为 `6P 3.81` 凤凰端子，由两组 `G / L / R` 构成，整体顺序为 `G / L / R / G / L / R`。每组默认按 `L-G`、`R-G` 两路非平衡信号使用。
- 口径差异：用户按功能称 `L / R / G`，Agent 按面板物理顺序称 `G / L / R`。最终采用规则为：背板按核实后的 6P 物理顺序绘制；逻辑分配严格采用用户确认的“两路信号共地”，不得把 L/R 当成平衡正负。
- 待继续核对项已由下方用户资料补齐：`LINE IN` 与 `LINE OUT` 均为 6 位端子，USB、主从级联、控制口和四路 SPK 的位置与用途均可从实物标注图重构。

### 主麦

- 主麦接本机电源，拨码置于 `M`；通过阵麦级联网线给从麦供电。
- 所有外部设备只连接主麦，不连接从麦。
- `EXT MIC IN` 接下一级从麦；`EXT MIC OUT` 接中控主机。
- USB-B 口用于 USB 音频输入、USB 音频输出和调试控制；资料要求使用配套 USB 线，敷设至调试电脑、会议平板或会议主机。
- 每次插拔级联网线后需重启主麦。
- `LINE OUT` 与 `LINE IN` 各为一组 6 位端子。每组可按两路平衡信号 `G / + / - / G / + / -` 使用，也可按四路非平衡信号 `G / 1 / 2 / G / 3 / 4` 使用。后一模式与用户确认的“两路信号共地”一致：`1 / 2` 共用第一个 `G`，`3 / 4` 共用第二个 `G`。
- SPK 为四个独立功放输出通道 `SP1-SP4`，每通道各占 `+ / -` 两孔，共 8 孔。

### 从麦

- 从麦拨码置于 `S`，由主麦或上一级阵麦通过级联网线供电，不接本机电源。
- `EXT MIC OUT` 接主麦或上一级从麦；`EXT MIC IN` 接下一级从麦。
- 从麦不连接其他外部设备；资料图明确禁用本机电源、LINE、SPK 等外部接线面。
- 每段阵麦级联网线长度不超过 `10m`。

### 主麦到中控的控制映射

- 设备侧接口为 RJ45，按 568B 线序取芯后转 RS232，不得建模为普通以太网数据链路。
- 白橙芯 `TX` 接 RS232 引脚 2 `RX`。
- 橙芯 `RX` 接 RS232 引脚 3 `TX`。
- 白绿 / 绿芯 `GND` 接 RS232 引脚 5 `GND`。
- 串口参数：波特率 `115200`，数据位 `8 bits`，起始位 `1 bit`，停止位 `1 bit`，校验 `none`，硬件流控 `none`。
